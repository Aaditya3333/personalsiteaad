# Personal Website

A beautiful, modern personal website built with FastAPI and TailwindCSS.

## Features

- 🚀 FastAPI backend with Jinja2 templating
- 🎨 Modern, responsive design with TailwindCSS
- 📱 Mobile-friendly navigation
- ✨ Smooth animations and transitions
- 🌟 Beautiful gradient effects
- 📧 Contact form with validation
- 🎯 Project showcase
- 📊 About section with skills

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python main.py
   ```

4. Open your browser and navigate to `http://localhost:8000`

## Project Structure

```
├── main.py              # FastAPI application
├── requirements.txt      # Python dependencies
├── templates/           # HTML templates
│   ├── base.html       # Base template
│   ├── index.html      # Home page
│   ├── about.html      # About page
│   ├── projects.html   # Projects page
│   └── contact.html    # Contact page
└── static/             # Static files (CSS, JS, images)
    ├── css/
    └── js/
```

## Customization

- Edit the templates to personalize your content
- Update colors and styles in `base.html`
- Add your own projects in `projects.html`
- Update contact information in `contact.html`

## Technologies Used

- **FastAPI** - Modern, fast web framework for building APIs
- **Jinja2** - Templating engine
- **TailwindCSS** - Utility-first CSS framework
- **Font Awesome** - Icon library
